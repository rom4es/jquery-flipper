jquery.flipper

Небольшой jquery плагин, который умеет переворачивать карточки

Wiew on Github

Опции

Имя Тип По умолчанию Описание
countCardsInRow Number 3 Количество карточек в строке
delayFlip Number 2000 Время между сменой следующей карточки (мс)
animateSpeed Number 1000 Скорость анимации смены карточки (мс)
animateType String "animFlip" Тип анимации.
"animFlip" - переворачивание, "animFadeIn" - исчезание/появление
random Boolean true Случайный порядок переворачивания карточек

Методы

Имя Аргументы Описание
start - Запустить плагин
stop - Остановить плагин
getOption option Получить значение параметра
setOption option, value Установить значение параметра
destroy - Уничтожить плагин


С него начать?

Подключить css и js

<link rel="stylesheet" type="text/css" href="jquery.flipper.css"/> <script type="text/javascript" src="jquery.flipper.js"></script>

Настроить Html-разметку

<div> class="container" <!--контейнер флиппера--> <div> <!--контейнер карточки--> <div></div> <!--передняя сторона карточки--> <div></div> <!--задняя сторона карточки--> </div> ... </div>

Инициализировать плагин

var plugin = $(".container").flipper({ // options }); plugin.flipper("setOption", "random", false); // Пример вызова метода

Пример 1 (дефолтные параметры)

var flipper1 = $(".b-card-list").flipper();


Пример 2

var flipper2 = $(".b-card-list-2").flipper({ countCardsInRow: 4, delayFlip: 1000, animateSpeed: 400, animateType: "animFadeIn", random: false, });
